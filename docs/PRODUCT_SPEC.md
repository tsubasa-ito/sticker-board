# シールボード — プロダクト仕様書

**アプリ名:** シールボード -デジタルシール帳-
**サブタイトル:** リアルシールをデジタルコレクション
**プラットフォーム:** iOS 18+ / iPhone

---

## 1. プロダクトの目的

> 現実のシールを撮影して、スマホの中に自分だけのシールボードを作れるアプリ

子どもの頃にシール手帳に夢中になった世代が、大人になった今もシール集めの楽しさを手軽にデジタルで体験できるプロダクト。

### 解決する課題

| 課題 | このアプリでの解決 |
|------|------------------|
| リアルなシールの保管が大変（かさばる・劣化する） | 撮影してデジタル保管 |
| シール手帳アプリは既製スタンプが中心 | 現実のシールをAIで切り抜いて使える |
| シールコレクションを映えるように見せたい | ボード上に自由にレイアウトして1枚の画像に |

## 3. 機能一覧

### シール撮影・切り抜き

| 機能 | 説明 |
|------|------|
| カメラ撮影 / カメラロール選択 | 写真をその場で撮るか、既存の写真から選択 |
| AI背景除去 | Vision Framework で被写体を自動切り抜き |
| 被写体の長押し選択 | 写真内のシールにしたい被写体を長押しで直接指定して切り抜き |
| マスク手動調整 | ブラシツールで切り抜き結果を微調整（消しゴム/復元、サイズ変更、Undo） |
| 複数シール一括検出 | 1枚の写真から複数のシールを個別に検出・選択・一括保存 |
| 撮影ガイド | きれいに切り抜くためのコツを表示（折りたたみ可能） |

### シールライブラリ

| 機能 | 説明 |
|------|------|
| グリッド一覧 | 保存したシールを最新順で表示 |
| プレビュー | タップで拡大表示 |
| 不要部分の再除去 | 長押しメニューからマスク再編集で不要部分を除去（上書き保存） |
| 向き変更（回転） | 長押しメニューから左/右に90度回転（上書き保存）。撮影時のプレビュー画面でも回転可能 |
| 削除 | 長押しで削除（使用中ボードの警告あり） |

### ボード編集

| 機能 | 説明 |
|------|------|
| ボード作成 | 新規作成＋初回起動時にデフォルトボード自動作成 |
| シール配置 | ドラッグ移動・ピンチ拡大縮小・2本指回転 |
| レイヤー操作 | 前面へ・背面へ・削除 |
| フィルター加工 | キラキラ・レトロ・パステル・ネオン・ぷっくり・ワッペンの6種＋オリジナル（配置単位で適用） |
| 枠線（ボーダー） | 太さ4段階 × 9色プリセット、輪郭に沿った自然な枠線 |
| 背景パターン | 無地・ドット・グリッド・ストライプ・グラデーションの5種、カラーカスタマイズ対応。Pro限定でカスタム写真を背景に設定可能 |
| ボード画像保存 | 完成ボードを1枚の画像として写真ライブラリに保存 |
| 自動保存 | 操作のたびにローカルに自動保存 |

### ナビゲーション

| 機能 | 説明 |
|------|------|
| 初回オンボーディング | 3ページのガイド（ウェルカム・撮影&切り抜き・ボード）でアプリの使い方を紹介。スキップ可能、ホーム画面から再表示可能 |
| ホーム画面 | カルーセル型ボード一覧（シール配置プレビュー付き） |
| フローティングタブバー | ホーム・撮影（+）・ライブラリの3タブ |
| 折りたたみ式ツールバー | ボード編集時のツールバーは折りたたみ可能 |
| アップデート通知 | 起動時にApp Storeの最新バージョンをチェックし、アラートで通知（24時間間隔、メジャー/マイナー区別、スキップ管理） |
| アプリ内レビュー訴求 | 達成感のあるタイミング（シール5/15/30枚目・ボード新規作成・起動5回目）でApple標準レビューダイアログを表示。90日クールダウン＋365日ローリングウィンドウで年3回上限 |

### ウィジェット

| 機能 | 説明 |
|------|------|
| ボードショーケースウィジェット | WidgetKit でお気に入りのボードをホーム画面に飾れるウィジェット（Medium / Large 対応、完全無料） |
| ボード選択 | ウィジェット長押し →「ウィジェットを編集」で表示するボードを選択（AppIntentConfiguration） |
| 自動更新 | ボード編集・保存時にスナップショットを自動生成しウィジェットに反映 |
| ディープリンク | ウィジェットタップで `stickerboard://board/{boardId}` → ボード編集画面に直接遷移 |
| データ共有 | App Group（`group.com.tebasaki.StickerBoard`）でスナップショット画像＋メタデータJSONを共有 |

## 4. デザイン方針

### 世界観

**「おしゃれな文房具屋さん」** — 温かみのあるクラフト紙のような質感に、ポップなアクセントカラーを添えたデザイン。

### カラーパレット

| 役割 | 色 | Hex |
|------|-----|-----|
| アクセント | ビビッドオレンジ | `#E87A2E` |
| セカンダリ | ディープネイビー | `#2A2D5B` |
| 背景 | クリーム | `#FAF0DE` |
| テキスト | ネイビー | `#2A2D5B` |

### UIルール

- **フラットカラー**: UIにグラデーションは使用しない（ボード背景パターンのグラデーション機能は例外）
- **iOS標準ナビゲーション**: 全画面で `.navigationTitle` + `.inline` スタイルを使用。カスタムトップバーは使わない
- **AppTheme集約**: 全色を `AppTheme` に定義。`Color(hex:)` の直指定は禁止
- **カードスタイル**: 角丸16pt + 薄い影のシール手帳風カード
