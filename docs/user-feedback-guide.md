# ユーザーフィードバックガイド

## 目的
価値の検証 - ユーザーにたくさん使ってもらい、課金につなげるためのインサイトを得る。

## 方法
非公開フィードバック謝礼方式（X/Twitter DM）

- App Store レビューの依頼は**しない**（Apple規約違反）
- SNS投稿を条件に**しない**（景品表示法・ステマ規制の回避）
- 謝礼: Amazonギフト券500円分（Eメールタイプ）

## DM テンプレート

```
こんにちは！突然のDM失礼します。

○○さんのシールの投稿、いつも楽しく拝見しています！
（※ 相手の投稿内容に合わせて一言カスタマイズする。例: 「コレクションの並べ方がすごくかわいくて」「レトロシールのセレクトが素敵で」など）

実は「シールボード」というiOSアプリを個人開発していまして、お気に入りのシールを撮影するとデジタルコレクションできるアプリです。

シールが本当にお好きな方に使っていただきたいと思っていて、○○さんにぴったりだと感じてご連絡しました。もしよければ、実際に使ってみた感想を聞かせていただけないでしょうか？ささやかですが、お礼にAmazonギフト券500円分をお送りしています。

アプリはこちらです：
（App Store URL）

教えていただきたいこと（気軽にで大丈夫です！）：
1. シールを撮影して切り抜けたとき、どう感じましたか？
2. 何枚くらい保存しましたか？もっと増やしたいと思いますか？
3. 「もっとこうしたいのにできない」と感じた場面はありましたか？
4. ボードの背景変更や枠線・フィルターは見ましたか？使いたいと思ったものはありますか？
5. 月額でPro機能が全部使えるとしたら、使いたいと思いますか？

番号ごとに一言ずつでもOKです！

※ App Storeレビューの依頼ではなく、開発改善のためのフィードバックです。よろしくお願いします！
```

### DM送信時のカスタマイズ例

相手の投稿内容を見て、冒頭の一言を変える。コピペ感をなくすのが返信率アップのポイント。

| 相手の特徴 | カスタマイズ例 |
|-----------|--------------|
| シールコレクション投稿が多い | 「コレクションの写真、いつも眺めてます！種類の多さがすごいですね」 |
| 手帳デコに使っている | 「手帳のシールの使い方がすごく参考になっていて」 |
| レトロ・昭和シール好き | 「レトロシールのセレクト、めちゃくちゃツボです」 |
| シール交換をしている | 「シール交換の投稿を見て、シール愛が伝わってきて」 |

## 質問の狙い

| # | 質問 | 狙い |
|---|------|------|
| 1 | 切り抜きの感想 | コア体験に感動があるか。ここが弱いと課金以前の問題 |
| 2 | 保存枚数・増やしたいか | エンゲージメント。コレクション欲が機能しているか |
| 3 | できないと感じた場面 | 無料制限（シール30枚/ボード1枚/枠線制限）に自然にぶつかっているか |
| 4 | Pro機能のプレビュー | 期待値駆動型ペイウォールが機能しているか |
| 5 | 課金意向 | 最終的な課金意向と理由 |

## 回答パターン別の打ち手

| Q1 | Q2 | Q3 | Q4 | Q5 | 状態 | やるべきこと |
|----|----|----|----|----|------|------------|
| 感動あり | 増やしたい | 壁に当たった | 使いたい | 課金したい | **理想** | 今の導線を磨く |
| 感動あり | 増やしたい | 壁に当たった | 興味薄い | 微妙 | **Pro機能の魅力不足** | Pro機能の価値を再設計 |
| 感動あり | 増やしたい | 当たってない | 見てない | わからない | **導線の問題** | 無料→有料の壁の見せ方を改善 |
| 感動あり | あまり | - | - | - | **リテンション不足** | 戻ってくる理由を作る |
| 微妙 | - | - | - | - | **コア体験の問題** | 切り抜き品質・UXの改善が最優先 |

## DM先の見つけ方

### ターゲット像
シールを日常的に集めている人、手帳・コラージュ・デコ文化が好きな人。

### 検索方法

#### 1. X/Twitter キーワード検索
- `シール集め` `シール交換` `シールコレクション`
- `フレークシール` `マスキングテープ` `コラージュ`
- `手帳デコ` `ほぼ日手帳 シール`
- `#シール沼` `#シール好きと繋がりたい` `#文具好き`

#### 2. ハッシュタグ探索
- `#シール交換` `#シールコレクション`
- `#文房具好き` `#手帳好きさんと繋がりたい`
- `#コラージュ` `#紙もの好き`

#### 3. コミュニティ・アカウント起点
- シール専門のショップアカウントのフォロワーを辿る
- シール交換募集ツイートのリプ欄から見つける
- 文具イベント（文具女子博 等）の参加報告ツイートから探す

### 優先順位（誰からDMすべきか）
1. **シールの写真を頻繁に投稿している人**（ヘビーコレクター = コア価値に共感しやすい）
2. **手帳やコラージュにシールを貼っている人**（ボード機能との親和性が高い）
3. **個人開発者・アプリ好き**（フィードバックの質が高い傾向）

### DM時の注意
- プロフィールに「DM開放」や「お気軽にDMください」がある人を優先
- DMが閉じている場合はリプライで「DMさせていただいてもいいですか？」と一声かける
- 一度に大量送信しない（スパム判定リスク）。1日5〜10件程度が目安
- 返信がなくても追いDMしない

## 目標
- **人数**: 10〜15人（多様な意見を得つつ管理可能な範囲）
- **予算**: 5,000〜7,500円
- **期間**: 2週間（DM送信1週間 + 回答待ち1週間）

## 法的注意事項
- App Store レビューの依頼は Apple 規約違反（絶対にしない）
- SNS投稿を報酬の条件にする場合は景品表示法のステマ規制に該当（PR表記が必要）
- 本ガイドの方式（非公開フィードバック謝礼）は上記に該当しない
