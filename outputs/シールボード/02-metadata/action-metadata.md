## メタデータ更新タスク（バージョン1.3.0提出時）

### サブタイトル更新

- [ ] apple-metadata.md の「サブタイトル最適化案A」を確認する
- [ ] App Store Connect > バージョン情報 > サブタイトル に入力する
  ```
  AIで切り抜き!推し活シールコレクション
  ```
- [ ] 文字数が30文字以内であることを確認する（18文字）

### キーワードフィールド更新

- [ ] apple-metadata.md の「最適化案」を確認する
- [ ] App Store Connect > バージョン情報 > キーワード に入力する
  ```
  シール帳,ステッカー,推し活,切り抜き,写真加工,コレクション,デコ,コラージュ,グッズ管理,シール交換,手帳,ウィジェット,背景除去,フィルター,レトロ,推しグッズ,シール整理
  ```
- [ ] 文字数が100文字以内であることを確認する（99文字）

### 説明文更新

- [ ] apple-metadata.md の「最適化案」の説明文をコピーする
- [ ] App Store Connect > バージョン情報 > 説明文 に貼り付ける
- [ ] プライバシーポリシーURL・利用規約URLが正しいことを確認する

### プロモーションテキスト更新

- [ ] apple-metadata.md の「通常時」プロモーションテキストをコピーする
- [ ] App Store Connect > バージョン情報 > プロモーションテキスト に貼り付ける
- [ ] 注: プロモーションテキストは審査なしで随時変更可能

### What's New 更新

- [ ] apple-metadata.md の What's New テキストをコピーする
- [ ] App Store Connect > バージョン情報 > このバージョンの新機能 に貼り付ける
