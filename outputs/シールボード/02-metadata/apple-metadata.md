# Apple App Store メタデータ最適化案 - シールボード

**作成日:** 2026-04-09
**プラットフォーム:** Apple App Store (日本)
**ステータス:** コピー&ペースト可能（そのまま App Store Connect に入力可能）

---

## 重要な前提

iTunes API の調査結果から、以下の問題が確認されました:

1. **言語設定が EN のみ** - 日本語ローカライゼーションが未設定の可能性
2. **スクリーンショットが0枚** - コンバージョンに致命的
3. **ストア上の説明文が docs/appstore-metadata.md と異なる** - 更新が反映されていない可能性
4. **ストア上のバージョンが 1.2.1** - 1.3.0 はまだ審査中または未提出

これらの問題を先に解決してから、以下のメタデータを適用してください。

### サブタイトル（30文字制限）

| | 内容 | 文字数 |
|---|------|:--:|
| **現在** | リアルシールをデジタルコレクション | 16文字 |
| **最適化案A** | AIで切り抜き!推し活シールコレクション | 18文字 |
| **最適化案B** | 撮って集めてデコる!推し活コレクション | 18文字 |
| **最適化案C** | シール撮影&AI切り抜き コレクション帳 | 18文字 |

**推奨: 案A**

理由:
- 「AI」: 検索トレンドワード。技術的優位性を即座にアピール
- 「切り抜き」: 背景除去の検索意図にマッチ
- 「推し活」: ターゲット層の主要キーワード（検索ボリューム高）
- 「コレクション」: 収集欲を刺激
- 現在の「リアルシールをデジタルコレクション」は意味は伝わるが、検索キーワードとしての威力が弱い

```
AIで切り抜き!推し活シールコレクション
```

## App Store Connect 設定の確認事項

### カテゴリ

| | 設定 |
|---|---|
| **プライマリカテゴリ** | 写真/ビデオ（iTunes APIで確認済み） |
| **セカンダリカテゴリ** | グラフィック/デザイン（iTunes APIで確認済み） |
| **推奨変更** | 変更不要。写真/ビデオは検索流入が多いカテゴリ |

注: ユーザーから提供された情報では「ライフスタイル」カテゴリとのことでしたが、iTunes APIの実データでは「Photo & Video」+ 「Graphics & Design」になっています。これは適切な設定です。

### ローカライゼーション（最重要）

**現状:** 英語（EN）のみ

**必要な対応:**
1. App Store Connect > アプリ情報 > ローカライゼーションで「日本語」を追加
2. 日本語のタイトル・サブタイトル・キーワード・説明文を登録
3. 日本語のスクリーンショットを登録
4. 日本向けのプロモーションテキストを登録
