# ビジュアルアセット仕様書 - シールボード

**作成日:** 2026-04-09
**プラットフォーム:** Apple App Store (日本)

---

## 緊急事項

**iTunes API の調査で、スクリーンショットが0枚であることが確認されました。**

App Store でアプリを閲覧するユーザーの約70%がスクリーンショットのみで判断すると言われています。スクリーンショットの登録は最優先事項です。

## スクリーンショット仕様

### 必須サイズ

| デバイス | サイズ (px) | 必須/任意 |
|---------|-----------|:--:|
| iPhone 6.9インチ (16 Pro Max) | 1320 x 2868 | 必須 |
| iPhone 6.3インチ (16 Pro) | 1206 x 2622 | 任意（6.9から自動生成可） |
| iPhone 6.1インチ (15) | 1179 x 2556 | 任意 |
| iPad Pro 13インチ | 2048 x 2732 | 推奨 |

### 推奨枚数

- **最低:** 5枚（App Store の必須要件）
- **推奨:** 8-10枚（スクロールして見るユーザーの離脱防止）
- **最大:** 10枚

## スクリーンショット構成案（8枚）

### 1枚目: ヒーローショット（ファーストインプレッション）

**キャプション:** 「シールを撮って、貼って、飾る」
**画面内容:** 完成されたボードの美しいスクリーンショット。カラフルなシールが配置されたボード
**デザインポイント:**
- アプリのカラーテーマ（ネイビー x オレンジ x クリーム）を背景に使用
- 完成ボードの実画面を大きく配置
- 上部に大きなキャッチコピー
- 下部にアプリアイコン

### 2枚目: AI切り抜き機能

**キャプション:** 「撮るだけでAIが自動切り抜き」
**画面内容:** カメラ撮影画面 → 切り抜き完了のビフォーアフター
**デザインポイント:**
- 左に元写真、右に切り抜き後のシール
- 矢印またはスパークルエフェクトで「AI処理」を表現
- 「タップするだけ!」等のバッジを追加

### 3枚目: ボード編集画面

**キャプション:** 「自由にレイアウト、あなただけのボード」
**画面内容:** ボード編集画面でシールを配置している様子
**デザインポイント:**
- ドラッグ中のシールを強調
- 回転・拡大縮小のUIを見せる
- 複数のシールが配置されたボード

### 4枚目: フィルター・デコレーション

**キャプション:** 「6種のフィルター&9色の枠線でデコ」
**画面内容:** フィルター選択画面。同じシールに異なるフィルターが適用された比較
**デザインポイント:**
- 6種類のフィルター名を表示（キラキラ、レトロ、パステル、ネオン、ぷっくり、ワッペン）
- ビフォーアフターまたは複数バリエーションを並べる

### 5枚目: 推し活ユースケース

**キャプション:** 「推し活グッズをデジタルコレクション」
**画面内容:** 推し活関連のシール（キャラクターグッズ、イベントステッカー等）を集めたボード
**デザインポイント:**
- ターゲット層（推し活ユーザー）に直接訴求
- ハートやスターなどのデコレーション要素
- 「保管場所に困らない!」等のバッジ

### 6枚目: ウィジェット機能

**キャプション:** 「ウィジェットでホーム画面に飾れる」
**画面内容:** iPhoneのホーム画面にシールボードウィジェットが表示されている画面
**デザインポイント:**
- 実際のiPhoneホーム画面のモックアップ
- ウィジェットが他のアプリアイコンと並んでいる様子
- 「いつでも推しと一緒」等のコピー

### 7枚目: シールライブラリ

**キャプション:** 「集めたシールをまとめて管理」
**画面内容:** シールライブラリ画面（グリッド表示）
**デザインポイント:**
- たくさんのシールが整然と並んでいる
- コレクション感を強調
- 「30枚まで無料!」等のバッジ

### 8枚目: プライバシー・安心感

**キャプション:** 「完全オフライン。データはあなたのiPhoneだけに」
**画面内容:** プライバシーを象徴するアイコン + アプリの安心ポイント
**デザインポイント:**
- Appleの「App Privacy」表示（データ収集なし）を引用
- 「ユーザー登録不要」「オフラインで動作」「AI処理はデバイス上」
- 信頼感を与えるクリーンなデザイン

## スクリーンショットデザインガイドライン

### カラーパレット（アプリテーマに準拠）

| 用途 | 色 | HEX |
|------|---|-----|
| メイン背景 | ネイビー | #2A2D5B |
| アクセント | オレンジ | #E87A2E |
| サブ背景 | クリーム | #FAF0DE |
| テキスト（明背景時） | ダークネイビー | #1A1D3B |
| テキスト（暗背景時） | ホワイト | #FFFFFF |

### フォント

- **キャプション:** 太めのゴシック体（Hiragino Sans W6 以上、または Noto Sans JP Bold）
- **サブテキスト:** 通常のゴシック体（Hiragino Sans W3）
- **サイズ:** キャプションはスクリーンショット幅の8-10%

### レイアウトルール

- キャプションは上部に配置（ユーザーが最初に目にする部分）
- アプリ画面は中央〜下部に配置
- 余白は上下左右に均等に確保
- デバイスフレーム（iPhoneの筐体）は使用してもしなくてもよい（トレンドはフレームなし）

## App Preview（動画）

**現在:** なし
**推奨:** 中期的に1本作成

### App Preview 仕様

| 項目 | 仕様 |
|------|------|
| 長さ | 15-30秒 |
| サイズ | スクリーンショットと同じデバイスサイズ |
| 形式 | H.264, AAC, .mov or .mp4 |
| 最大容量 | 500MB |

### 推奨構成（30秒）

1. (0-5秒) オープニング: シールの撮影
2. (5-12秒) AI切り抜きのデモ: 撮影 → 切り抜き完了
3. (12-20秒) ボード編集: シール配置 → フィルター適用 → 枠線追加
4. (20-27秒) 完成ボードの表示 → ウィジェットでホーム画面に表示
5. (27-30秒) CTA: 「今すぐ無料でダウンロード」
